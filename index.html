How does JavaScript Work ?

JavaScript is a single threaded, interpreted, lightweight and the world most popular programming language. It works with own JIT compiler of every web browsers. JIT is abbreviated form of Just In Time. Basically in JavaScript JIT is such a process where engine convert JavaScript code to machine code (bytecode). So that machine easily understand JavaScript Code and he can delivary more faster output to the user. Like, the Google Chrome browser uses a JIT engine called V8 for convert JavaScript code to bytecode.

What does JS Event Loop ?

JavaScript basically handles one operation at a time, or one line of code at a time. It is capable of working both sychronously and asychronously. Sychoronous tasks are run relatively before JavaScript and Asychronous tasks are stored in Queue. The function of Event Loop is to send the waiting codes in this Queue to Stack in time to run.
